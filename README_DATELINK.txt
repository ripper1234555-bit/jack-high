================================================================================
QUICK START (3 STEPS)
================================================================================

1. Extract the ZIP:
   unzip DateLink_Dating_App.zip

2. Install dependencies:
   pip install -r requirements.txt
   
   OR run setup script:
   ./setup.sh (Mac/Linux)
   setup.bat (Windows)

3. Run the app:
   streamlit run dating_app.py

DONE! Open: http://localhost:8501

================================================================================
GITHUB CODESPACE
================================================================================

1. Push to GitHub:
   cd DateLink_Dating_App
   git add .
   git commit -m "Add DateLink app"
   git push

2. Open Codespace:
   Go to GitHub repo → Code → Codespaces → Create codespace

3. In Codespace terminal:
   pip install -r requirements.txt
   streamlit run dating_app.py

4. Click the Streamlit URL to open the app

DATA STORAGE
================================================================================

Data persists between app restarts.
